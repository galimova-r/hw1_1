Реализовать рест сервис для проекта TODO-лист. У проекта 3 сущности Пользователь, Задание, Категория.

 
Пользователь имеет следующие атрибуты:

Уникальный идентификатор
Логин
Фамилия
Имя
Отчество
Дата рождения
 
Задание имеет следующие атрибуты:

Уникальный идентификатор
Наименование
Описание
Дата выполнения
Выполнено (значение Да/Нет)
Категория
Пользователь
 
Категория имеет следующие атрибуты:

Уникальный идентификатор
Наименование
 
Общие атрибуты для всех сущностей:

Дата создания
Дата обновления
 
Связи:

Пользователь – Задание – один ко многим

 
Задание – Категория – многие ко многим

 
В задании можно использовать любую базу данных.
